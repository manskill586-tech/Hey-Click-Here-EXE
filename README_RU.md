Визуальная новелла‑эксперимент с редактором сцен, системными эффектами и фейковым терминалом. Проект рассчитан на запуск через Electron и поддерживает локализацию.

## 1) Что это за проект
- Формат: визуальная новелла (Web + Electron).
- Фокус: история + механики (таймлайн, эффекты текста, системные иллюзии).
- Языки: RU / EN.
- Режимы: игра, редактор, фейковый CMD.

## 2) Интерфейс игры
- Сцена: фон + персонажи.
- Диалог: имя говорящего + текст.
- Выборы: кнопки вариантов.
- Быстрое меню: Back / Auto / Skip / Save / Load / Settings.

## 3) Управление
- Enter / Space — продолжить.
- 1–9 — выбрать вариант.
- Ctrl+1…3 — сохранить.
- Alt+1…3 — загрузить.
- A — Auto.
- S — Skip.
- B — Back.
- O — Settings.
- L — Log.
- M — Mute.
- H — Hide UI.
- C — Checkpoints.
- Esc — закрыть панели или вернуть UI.

## 4) Настройки
- Text: скорость, размер, прозрачность текста.
- Audio: громкости музыки, голоса, эффектов, эмбиента.
- Gameplay: авто‑задержка, таймер выбора, skip unread.
- System: эффекты, давление, смена языка, утилиты.

Выбор языка появляется при запуске и доступен в настройках.

## 5) Быстрый старт
1. Установить зависимости:
```
npm install
```
2. Запуск игры:
```
npm run dev
```
3. Запуск редактора:
```
npm run editor
```
4. Веб‑режим:
```
npm run web
```

## 6) Редактор сцен
Редактор позволяет:
- добавлять и редактировать сцены;
- управлять персонажами, позициями и эффектами;
- собирать таймлайн событий;
- тестировать сцену напрямую.

## 7) Сюжет и локализация
- Основной сюжет хранится в `story.js`.
- Переводы лежат в `story.js → locales`.
Пример:
```
locales: {
  en: { scenes: { "1": { text: "Hello!" } } }
}
```
Если локалей нет — текст остаётся в оригинале.

## 8) Фейковый CMD
Поддерживает сценарии через таймлайн:
- terminal.open
- terminal.print
- terminal.type
- terminal.choice
- terminal.close

В редакторе есть вкладка CMD и шаблоны для быстрого набора.

## 9) Сборка EXE
```
npm run dist
```
Готовый установщик лежит в `dist/`.

## 10) Структура проекта
- index.html, game.js, styles.css — игра.
- story.js — сюжет и сцены.
- characters.js — библиотека персонажей.
- editor.html/.js/.css — редактор.
- cmd.html/.js — фейковый терминал.
- credits.* — титры.

## 11) Известные нюансы
- Аудио не стартует до первого клика (ограничение браузеров).
- Хромакей требует http/https, не file://. Используй:
```
npx serve .
```
- Dev‑инструменты включаются при наличии `dev.key` в корне.
- Если сборка ругается на кэш NSIS, очисть:
```
Remove-Item -Recurse -Force "$env:LOCALAPPDATA\electron-builder\Cache\nsis"
```
